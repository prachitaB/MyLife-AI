# MyLife AI

MyLife AI is a Human-in-the-loop AI Life Operating System that helps users manage goals, milestones, tasks, journaling, and productivity tracking using AI + automation.

---

## 🚀 Tech Stack
- n8n (Automation Backend)
- OpenAI AI Agents
- Google Sheets (Database)
- HTML + Tailwind + Vanilla JS (Frontend)
- GitHub (Version Control)

---

## 🧠 Features
- AI Goal Planner
- Milestone Generator
- Task Generator
- Daily Journal Analyzer
- AI Coach System
- Productivity Dashboard (WIP)

---

## 📁 Project Structure
frontend/ → UI  
workflows/ → n8n automation flows  
docs/ → project documentation  

---

## ⚙️ Setup
1. Clone repo
2. Open frontend/index.html using Live Server
3. Import workflows into n8n
4. Connect Google Sheets
5. Update webhook URLs in frontend

---

## 🔗 Webhooks
- Goal Planner → /goal-planner  
- Task Generator → /task-generator  
- Journal → /journal  
- Coach → /coach
